Button 按钮

按钮用于开始一个即时操作。

何时使用#

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

代码演示

  • link按钮
按钮类型

按钮有四种类型：默认按钮、gray按钮、light按钮、link按钮、clear按钮。

通过设置 class 为 btn btn-gray btn-light btn-link btn-clear 可分别创建按钮，若不设置属性 class 值则为默认按钮。

当按钮处于不可用状态时,只需要将属性 class值中增加 disabled或增加属性 disabled

<button class="btn">默认按钮</button> <button class="btn btn-gray">gray按钮</button> <button class="btn btn-light">light按钮</button> <button class="btn btn-link">link按钮</button> <button class="btn-clear">clear按钮</button>
图标按钮

图标按钮有三种类型：默认图标按钮、gray图标按钮、clear图标按钮。

通过设置 class 为 btn btn-icon-primary btn btn-icon-gray btn btn-icon-clear 可分别创建不同类型的图标按钮。

通过设置 class 为 btn-icon-lg btn-icon-sm 分别把按钮设为大、小尺寸。若不设置，则尺寸为默认。

<button class="btn btn-icon-primary"> <i class="qIcon youjian"></i> </button> <button class="btn btn-icon-gray"> <i class="qIcon guanbi"></i> </button> <button class="btn btn-icon-clear"> <i class="qIcon banshou"></i> </button> <button class="btn btn-icon-primary btn-icon-lg"> <i class="qIcon shouji"></i> </button> <button class="btn btn-icon-primary"> <i class="qIcon shouji"></i> </button> <button class="btn btn-icon-primary btn-icon-sm"> <i class="qIcon shouji"></i> </button>
按钮尺寸

按钮有大、默认、小三种尺寸。

通过设置 class 为 btn-lg btn-sm 分别把按钮设为大、小尺寸。若不设置，则尺寸为默认。

<button class="btn btn-lg">大按钮</button> <button class="btn">默认尺寸</button> <button class="btn btn-sm">小按钮</button>
含图标的按钮

含图标的按钮可直接在按钮广本中增加图标标签即可

<button class="btn"><i class="qIcon zhuce"></i>默认按钮</button> <button class="btn">默认<i class="qIcon jiahao"></i>按钮</button> <button class="btn">默认按钮<i class="qIcon xiazai"></i></button>
