Checkbox 多选框

多选框。

  • 在一组可选项中进行多项选择时；
  • 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

代码演示

内联多选框

默认为内联样式,通过设置label标签的属性 class值为.checkbox, 实现自定义多选框功能。

多选框的文字说明在右侧时,可使用标签span设置属性class 值为.label-right可适当增加左右边距。

<label class="checkbox-inline">
<div class="checkbox"> <input class="input-checkbox" value="2" type="checkbox"> <span class="style-checkbox"></span> </div> <span class="label-right">单选三</span> </label>
多选框列表

多选框列表样式是通过在label标签的父节点设置属性 class值为.checkbox-block, 实现自定义多选框功能。

<label class="checkbox-block">
<div class="checkbox"> <input class="input-checkbox" value="2" type="checkbox"> <span class="style-checkbox"></span> </div> <span class="label-right">选项内容</span> </label>
单选框

通过使用标签 label ,配合 class名 实现自定义单选框功能。

<label class="checkbox-inline checkbox-disabled">
<div class="checkbox"> <input class="input-checkbox" disabled value="2" type="checkbox"> <span class="style-checkbox"></span> </div> <span class="label-right">选项内容</span> </label>
纯样式多选框

纯样式多选框是通过在label标签或其父节点设置属性 class值为.checkbox-disabled和.checkbox-checked实现的。

<label class="checkbox-block checkbox-checked checkbox-disabled">
<div class="checkbox"> <input class="input-checkbox" disabled value="2" name="checkbox1" type="checkbox"> <span class="style-checkbox"></span> </div> <span class="label-right">模拟多选框样式</span> </label>
