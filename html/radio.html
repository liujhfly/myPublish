Radio 单选框

单选框。

  • 用于在多个备选项中选中单个状态。
  • 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

代码演示

内联单选框

默认为内联样式,通过设置label标签的属性 class值为.radio, 实现自定义单选框功能。

单选框的文字说明在右侧时,可使用标签span设置属性class 值为.label-right可适当增加左右边距。

<label class="radio-inline">
<div class="radio"> <input class="input-radio" value="2" name="radio1" type="radio"> <span class="style-radio"></span> </div> <span class="label-right">单选三</span> </label>
单选框列表

单选框列表样式是通过在label标签的父节点设置属性 class值为.radio-block, 实现自定义单选框功能。

<label class="radio-block">
<div class="radio"> <input class="input-radio" value="2" name="radio1" type="radio"> <span class="style-radio"></span> </div> <span class="label-right">单选三</span> </label>

不可用状态单选框

单选框不可用状态是通过在label标签的父节点设置属性 class值为.radio-disabled, 并且需要在隐藏的input标签中增加disabled属性。

<label class="radio-inline radio-disabled">
<div class="radio"> <input class="input-radio" disabled value="2" name="radio1" type="radio"> <span class="style-radio"></span> </div> <span class="label-right">单选三</span> </label>
纯样式单选框

纯样式单选框是通过在label标签或其父节点设置属性 class值为.radio-disabled和.radio-checked实现的。

<label class="radio-block radio-checked radio-disabled">
<div class="radio"> <input class="input-radio" disabled value="2" name="radio1" type="radio"> <span class="style-radio"></span> </div> <span class="label-right">模拟单选框样式</span> </label>
